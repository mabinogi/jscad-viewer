A viewer for JSCad designs.

The end goal is a template + package combination to easily preview multiple jscad designs in the same repo without too much local boilerplate to maintain.
Initial steps will be to just get the rendering working, along with parameter support and design selection.

Splitting it up and packaging can come later - for my own projects I'll probably just fork this repo.


Inspiration - https://github.com/simplyhexagonal/create-jscad-project

I'd have just extended and contributed to that, except that it's written in Vue, and I'm a React developer,
so I figured it would be easier to start from scratch than get my head around Vue.
